## 📁 ЧТО БЫЛО СОЗДАНО

### 1️⃣ Папка `functions/` с двумя файлами:

**`functions/package.json`**
- Конфигурация зависимостей Node.js
- Команды для локального тестирования и развёртывания

**`functions/index.js`**
- ✅ `sendBookingNotification()` — отправка заявок в Telegram
- ✅ `sendRegistrationNotification()` — уведомление о регистрации
- ✅ Валидация данных на сервере
- ✅ Rate limiting (максимум 3 заявки в минуту)
- ✅ Сохранение данных в Firestore коллекцию `booking_submissions`
- ✅ Безопасное хранение Telegram токена в переменных окружения

### 2️⃣ Обновлённый `index.html`

**Импорты:**
```javascript
import { getFunctions, httpsCallable } from "firebase-functions.js";
```

**Регистрация пользователя:**
- Вызывает `sendRegistrationNotification()` при новой регистрации
- Не блокирует регистрацию если Telegram недоступен

**Форма бронирования:**
- Вызывает `sendBookingNotification()` при отправке
- Показывает статус отправки пользователю
- Обрабатывает ошибки красиво

## 🚀 БЫСТРЫЙ СТАРТ (5 шагов)

### 1. Перевыпустить токен Telegram
```
Telegram → @BotFather → /revoke → выбрать бота → /token → копировать
```

### 2. Установить зависимости
```bash
cd functions
npm install
```

### 3. Установить переменные окружения
```bash
cd /Users/maksimurcenkov/Documents/GitHub/Vector
firebase functions:config:set telegram.token="НОВЫЙ_ТОКЕН" telegram.chat_id="-1003612200808"
```

### 4. Развернуть функции
```bash
firebase deploy --only functions
```

### 5. Тестировать формы на сайте
```bash
python3 -m http.server 8000
# Открыть: http://localhost:8000
# Зарегистрироваться и отправить заявку
```

## 📞 ПОДДЕРЖКА

Если что-то не работает:

1. **Ошибка авторизации Firebase CLI**
   ```bash
   firebase logout && firebase login
   ```

2. **Функция не развёрнута**
   ```bash
   firebase deploy --only functions
   ```

3. **Telegram не получает сообщения**
   ```bash
   firebase functions:config:get
   # Проверить токен и chat_id
   ```

4. **Смотреть логи в реал-тайме**
   ```bash
   firebase functions:log --tail
   ```
